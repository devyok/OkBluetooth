# OkBluetooth
Android蓝牙音频及消息通信框架

## 作用 ##
- OkBluetooth主要帮助应用与蓝牙设备之间建立可靠的HFP或SPP连接,并同时建立稳定的SCO音频连接；
- OkBluetooth会根据当前手机终端所连接的音频输出设备(蓝牙耳机/有线耳机/扬声器等)的状态来选择音频设备放音；
- 可以使用OkBluetooth提供的接口,搭建自己的音频及消息的通信组件；
